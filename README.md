## Countdown 倒數計時器
---
#### 練習目標
- 以純 vanilla JS 開發
- api載入資料並判定有效區間，再決定是否採用
- 實作倒數計時功能, 包括: 
1. Start button 按下可啟動計時器
2. Stop button 暫停計時器
3. 再次按下Start button 按下可重新啟動計時器
4. 啟動中 Start Button 為 disabled
5. 停止中 Stop Button 為 disabled
